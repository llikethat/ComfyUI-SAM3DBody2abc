# OPTIONAL: Native Alembic Export
# ============================================================

#
# To install native Alembic:
#   conda install -c conda-forge alembic
#   OR
#   pip install PyAlembic (may need to build from source)
#
# ============================================================
# OPTIONAL: FBX Export
# ============================================================
# FBX export requires Blender to be installed on your system.
# The extension will automatically find Blender if it's in
# standard installation locations or in PATH.
#

#
# ============================================================

#

#   pip install -e "git+https://github.com/cvg/GeoCalib#egg=geocalib"
#
